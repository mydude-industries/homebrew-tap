# homebrew-tap

The Homebrew tap for [pushto.host](https://pushto.host).

## Install the CLI

```
brew install mydude-industries/tap/pushto
```

That's it. Then run `pushto login` to get going.

## What is this repo?

A place that holds a Ruby file (`Formula/pushto.rb`) and some tarballs. Homebrew
uses it to install the `pushto` CLI on your Mac or Linux box without
needing Node.js, npm, or any of that.

The formula is auto-generated from the main pushto repo whenever we cut a
release, so please don't send PRs here — any changes will get overwritten on
the next release. If something's broken with the install,
[open an issue on pushto](https://github.com/mydude-industries/pushto/issues)
instead.
